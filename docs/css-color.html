Color

You can use those classes to change the background and fill colors of an element such as text, icons and svg elements.

Class Description
bk-color-fill--negative Negative fill color.
bk-color-fill--critical Critical fill color.
bk-color-fill--positive Positive fill color.
bk-color-fill--informative Informative fill color.
bk-color-bg--negative Negative background color.
bk-color-bg--critical Critical background color.
bk-color-bg--positive Positive background color.
bk-color-bg--informative Informative background color.
bk-color--negative Negative text color.
bk-color--critical Critical text color.
bk-color--positive Positive text color.
bk-color--informative Informative text color.
