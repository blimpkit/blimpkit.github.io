Vertical Navigation

The vertical navigation is used as a top-level application navigation list.

Attributes - Vertical Nav

Name Type Description
condensed boolean Condensed navigation mode.
can-scroll boolean Enable/disable scroll navigation support. Default is false.

Attributes - Vertical Nav Item

Name Type Description
indicated boolean Navigation item is indicated.
expandable boolean The item can expand.
is-expanded boolean If the item is expanded or not.

Attributes - List Navigation Item Popover

Name Type Description
is-expanded boolean If the popover is expanded or not.

Attributes - List Navigation Item Arrow

Name Type Description
is-expanded boolean If the popover is expanded or not.

Attributes - List Navigation Item Icon

Name Type Description
glyph string Icon class. For example 'sap-icon--home'.
svg-path string The src path to your svg image.
icon-size string The size of the item icon. Possible options are include 'lg' and none.
