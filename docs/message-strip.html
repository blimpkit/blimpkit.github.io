Message Strip

Message strips are used to draw the user`s attention to information that is important in the context of a page.

Attributes

Name Type Required Description
glyph string True Icon class
state string False Optional semantic state. Could be one of 'success', 'error', 'warning' or 'information'
on-dismiss function False Makes the strip dismissable and will trigger the provided function. Similar to "ng-click".
