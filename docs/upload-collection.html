Upload collection

The upload collection control allows users to upload single or multiple files.

Attributes - Upload Collection

Name Type Required Description
small boolean False Whether or not this is the small upload collection.
selection boolean False Whether or not this upload collection supports selection.

Attributes - Upload Collection Item

Name Type Required Description
selected boolean False Whether or not this item is selected
file-name boolean True The name of the file, not including the type extension.
extension boolean False The file type extension.
file-name-changed function False Event emitted when the user changes a file name. Args: (fileName : String)
delete-clicked function False Event emitted when presses the delete button.
editable boolean False Whether or not this upload collection item supports filename editing.
