Pagination

Pagination allows users to separate their content into discrete pages, making it easier to digest and navigate through. Pagination is commonly used for tables, card grids and tiles.

Attributes

Name Type Required Description
total-items number True The total number of the items
items-per-page number False The number of the items per page
current-page number False The number of the current page (starting from 1).
compact boolean False Pagination buttons size
display-total-items boolean True Whether to display the total number of items
items-per-page-options array<number> True The options for items per page dropdown. If not specified the dropdown will not be displayed
page-change function True Callback called when the page has changed. Args: (pageNumber : Number)
items-per-page-change function True Callback called when the 'itemsPerPage' dropdown selection has changed: Args: (itemsPerPage : Number)
items-per-page-placement string True Placement of the dropdown for items per page. See 'placement' on fdSelect.
