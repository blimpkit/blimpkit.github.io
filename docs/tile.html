Tile

Tiles can provide shortcuts to pages and modules. They can also be used as previews or to group related information in a container.

Attributes - Tile Container

Name Type Description
no-padding boolean Removes the default padding.
no-wrap boolean Do not flex wrap inner elements.
is-list boolean Contains list-like items.

Attributes - Tile

Name Type Description
is-small boolean Tile small size mode.
is-long boolean Double tile width.
is-slide boolean Slideshow tile mode.
is-line boolean Line mode. Use only with 'bk-tile-container'.
is-action : boolean Action mode. Use only with 'bk-tile-container'.

Attributes - Tile Background

Name Type Description
link string Tile background image URL.

Attributes - Tile Page Indicator

Name Type Description
pages number Number of pages.
active-page number The index number of the ictive dot indicator.

Attributes - Tile Header

Name Type Description
title string Tile header title.
subtitle string Tile header subtitle.
two-columns boolean Two column mode.
