Dialog

The dialog component is a container that appears in response to an action made by the user. It interrupts the app's current process to prompt the user for either information or a confirmation, which requires them to make a decision before the process can continue.

Attributes - Dialog

Name Type Description
visible boolean Show/hide dialog.
window-width string The width of the window. This maps to the CSS width property.
window-height string The height of the window. This maps to the CSS height property.
window-max-width string The max width of the window. This maps to the CSS max-width property.
window-max-height string The max height of the window. This maps to the CSS max-height property.
window-min-width string The min width of the window. This maps to the CSS min-width property.
window-min-height string The min height of the window. This maps to the CSS min-height property.

Attributes - Dialog Header

Name Type Description
compact boolean Applies compact style to the header and all elements inside the it.
header string Header text.
title string Dialog title text.
subheader string Subheader text.

Attributes - Dialog Footer

Name Type Description
compact boolean Applies compact style to the footer and all elements inside the it.
