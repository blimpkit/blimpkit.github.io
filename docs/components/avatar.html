Avatar

The avatar control is used for presenting various images, including user profiles, user initials, placeholder images, icons, or business-related images like product pictures.

Attributes

Name Type Required Description
is-placeholder boolean False The avatar will have a placeholder background.
is-transparent boolean False The avatar will have a transparent background.
interactive boolean False The avatar is clickable.
size string False The size of the avatar. Possible options are 'xs', 's', 'm', 'l' and 'xl'. Default is 'xs'.
glyph string False Icon class.
border boolean False Show border around avatar.
circle boolean False Avatar is in a circular shape.
zoom-icon string False Icon class.
zoom-label string False ARIA description of the zoom button.
image string False Link to an image.
