Step Indicator

The step indicator provides orientation and progress feedback during flows like onboarding, checkout, form completion or setup wizards.

This is NOT a navigational component. Its purpose is to be a non-interactive visual indicator.

Attributes

Name Type Required Description
steps array True An array of step objects
current-step number True Current step starting from 1
icon-only boolean False Hide step labels
type string False The type of the indicator. Possible options are 'bar' or none.

Examples

$scope.steps = [{ label: 'Step 1', glyph: 'sap-icon--product' },{ label: 'Step 2', glyph: 'sap-icon--cart' }, { label: 'Step 3', secondaryLabel: 'Current step', svg: './logo/blimpkit-symbolic.svg', }, { label: 'Step 4', glyph: 'sap-icon--wallet' }]; $scope.currentStep = 3;
