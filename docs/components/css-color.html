Color

You can use those classes to change the background and fill colors of an element such as text, icons and svg elements.

Fill colors

Class Description
bk-color-fill--negative Negative fill color.
bk-color-fill--critical Critical fill color.
bk-color-fill--positive Positive fill color.
bk-color-fill--informative Informative fill color.
bk-color-fg-fill--negative Negative foreground fill color.
bk-color-fg-fill--critical Critical foreground fill color.
bk-color-fg-fill--positive Positive foreground fill color.
bk-color-fg-fill--informative Informative foreground fill color.

Class Description

bk-color-bg Background color.
