Tool header

The tool header is the uppermost section of an application. It is always visible to give users access to various functions.

Attributes - Tool Header

Name Type Required Description
has-menu boolean False If the toolbar will contain a hamburger menu.
size string False Manually set the horizontal paddings of the tool header. Possible options are 'sm', 'md', 'lg' and 'xl'.

Attributes - Tool Header Group

Name Type Required Description
has-menu boolean False The group will have a menu inside it.
position string False Position of the group - 'center' or 'right'.
is-hidden boolean False Set the group to be hidden.
