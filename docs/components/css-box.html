Box

BlimpKit provides a horizontal, vertical and centered content box as well as some utility classes for managing flex containers.

Boxes

Class Description
bk-vbox Vertical flex layout.
bk-hbox Horizontal flex layout.
bk-center Flex layout with centered content.
bk-center--horizontal Flex layout with horizontally centered content.
bk-center--vertical Flex layout with vertically centered content.

Wrap

Class Description
bk-flex--wrap Box items will wrap on overflow.
bk-flex--nowrap Box items will never wrap.

Grow or shrink

Class Description
bk-box--items-stretch Grow items inside a container.
bk-stretch Grow item.
bk-shrink Shrink item.

Item Placement

Class Description
bk-box--center Items will centered in the container.
bk-box--end Items will be placed at the end of the container.

Justify Content

Class Description
bk-box--space-around Justify items along the container's main axis such that there is an equal amount of space on each side of each item.
bk-box--space-between Justify items along the container's main axis such that there is an equal amount of space between each item.
bk-box--space-evenly Justify items along the container's main axis such that there is an equal amount of space around each item, but also accounting for the doubling of space you would normally see between each item when using bk-box--space-around.

Align Items

Class Description
bk-box--items-start Align items to the start of the container's cross axis.
bk-box--items-center Align items along the center of the container's cross axis.
bk-box--items-end Align items to the end of the container's cross axis.

Gap

You can set a gap between items in boxes and grids.

Class Description
bk-box--gap-none No gap
bk-box--gap Gap of 0.5rem
bk-box--gap-m Gap of 1rem
bk-box--gap-l Gap of 1.5rem
bk-box--gap-xl Gap of 2rem
bk-box--gap-xxl Gap of 3rem
