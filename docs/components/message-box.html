Message Box

The message box displays a dialog with a simple message to the user. This component inherits the look and behavior of the Dialog.

Attributes - Messahe Box

Name Type Required Description
visible boolean True Show/hide message box.
title string True Message box title.
type string False Semantic type - 'confirmation', 'error', 'success', 'warning' and 'information'

Attributes - Messahe Box Footer

Name Type Required Description
compact boolean False Applies compact style to the footer and all elements inside the it.
