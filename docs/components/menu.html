Menu

The menu component is the listing structure with optional headers to create menus. It can also be used as the contents of dropdowns, context menus, etc, when paired with the popover component.

Attributes - Menu

Name Type Required Description
max-height number Maximum height in pixels before it starts scrolling. Default is the height of the window.
can-scroll boolean False Enable/disable scroll menu support. Default is false.
has-icons boolean False Enable/disable menu item icon support. Default is false.
show boolean False Use this instead of the CSS 'display' property. Otherwise, the menu will not work properly. Default is true.
no-backdrop boolean False Disables the backdrop. This may break the menu if not used properly. Default is false.
no-shadow boolean False Removes the shadow effect. Default is false.
close-on-outer-click boolean False Hide the menu when a user clicks outside it Default is true.

Attributes - Menu Item

Name Type Required Description
title string True Menu item label.
shortcut string False Secondary text. Most often used as a shotcut hint.
has-separator boolean False The menu item will have a separating line on the bottom side.
is-active boolean False Set the menu item as active.
is-selected boolean False Set the menu item as selected.
is-disabled boolean False Set the menu item as disabled.
left-icon-class string False Icon class. Displays the icon before the title.
left-icon-path string False Icon url path. Preferably an svg. Shown before the title. Alternative to leftIconClass.
right-icon-class string False Icon class. Displays the icon at the end of the menu item.
right-icon-path string False Icon url path. Shown after the title. Alternative to rightIconClass.
link string False URL string. Maps to href.

Attributes - Menu Sublist

Name Type Required Description
title string True Title/label of the menu item.
has-separator boolean False The menu item will have a separating line on the bottom side.
max-height number False Maximum height in pixels before it starts scrolling. Default is the height of the window.
can-scroll boolean False Enable/disable scroll menu support. Default is false.
is-disabled boolean False Set the menu sublist as disabled.
icon-class string False Icon class. Displays the icon before the title.
icon-path string False Path to an icon. Alternative to iconClass.
