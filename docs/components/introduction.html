What is BlimpKit?

BlimpKit is a UI component library for AngularJS built on top of the fundamental-styles CSS framework.

Although it generally adheres to the fundamental-styles design guidelines, there are some intentional differences. Certain components have been modified or removed, so BlimpKit should not be considered an official implementation of fundamental-styles.

BlimpKit is an independent library and also includes additional components that are unique to the project.

Getting Started

Install from NPM:

npm install @blimpkit/blimpKit

Install webjar using maven:

<dependency> <groupId>org.webjars.npm</groupId> <artifactId>blimpkit__blimpkit</artifactId> <version>{{version}}</version> </dependency>

Use CDN:

<script type="text/javascript" src="https://unpkg.com/@blimpkit/blimpkit@{{version}}/dist/blimpkit.min.js"></script>

Dependencies

You will also need AngularJS, jQuery, fundamental-styles and SAP Theming:

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular-aria.js"></script>

Base styles and themes

Themes are supported. There are currently three official themes available: BlimpKit, Classic and Mystic.

The BlimpKit and Classic themes provide light, dark, and auto modes. The Mystic theme supports light mode only.

To apply a theme, replace "themeName" with the desired theme. The available options are:

  • blimpkit-auto
  • blimpkit-light
  • blimpkit-dark
  • classic-auto
  • classic-light
  • classic-dark
  • mystic-light

For light or dark:

<link href="https://unpkg.com/fundamental-styles@0.38.0/dist/fundamental-styles.css" rel="stylesheet"> <link href="https://unpkg.com/@blimpkit/blimpkit@{{version}}/dist/css/themes/{themeName}.css" rel="stylesheet"> <link href="https://unpkg.com/@blimpkit/blimpkit@{{version}}/dist/css/themes/{themeName}-variables.css" rel="stylesheet"> <link href="https://unpkg.com/@blimpkit/blimpkit@{{version}}/dist/css/blimpkit.css" rel="stylesheet">

For auto:

<link href="https://unpkg.com/fundamental-styles@0.38.0/dist/fundamental-styles.css" rel="stylesheet"> <link href="https://unpkg.com/@blimpkit/blimpkit@{{version}}/dist/css/themes/{themeName}-auto.css" rel="stylesheet"> <link href="https://unpkg.com/@blimpkit/blimpkit@{{version}}/dist/css/blimpkit.css" rel="stylesheet">

Initialization and use

Add BlimpKit to your module as a dependency:

angular.module('yourModuleName', ['blimpKit']);

That's it! You can now use BlimpKit. For examples, look under the "Components" section in the sidebar.
