Button

The button component is used to trigger an action.

Attributes - Button

Name Type Required Description
label string True Button text.
compact boolean False Button size.
badge string/number False Used for showing a badge inside the button.
glyph string False Icon class/classes.
glyph-rotate number False Rotate glyph X degrees, where X can be 90, 180, and 270.
disabled-focusable boolean False Set the disabled but focusable state. This must always be used with the 'instructions' attribute.
toggled boolean False Set the toggle state.
type string False 'emphasized', 'transparent', 'ghost', 'positive', 'negative' and 'attention'. If not specified, normal state is assumed.
instructions string False Short description when the button is disabled. It should contain the reason and what needs to be done to enable it. This is for screen readers.
is-menu boolean False Adds an arrow to the button.
arrow-direction string False Direction of the menu arrow. Possible options are "up", "left", "right" and "down" (default).
is-overflow boolean False Used when the button is in a toolbar overflow popover.
is-split boolean False (Internal use) If the button is part of a split button.
in-group boolean False If the button is inside an bk-input-group-addon element.
in-msg-strip boolean False If the button is inside a message strip (see bk-message-strip).
nested boolean False Nested button mode.
decisive boolean False Indicates that the button is decisive. Used for dialog actions.
round boolean False Make the button round. Works only for icon buttons with no label.

Attributes - Split Button

Name Type Required Description
main-action string False Main button text
main-glyph string False Icon class for the button.
compact boolean False Button size.
glyph string False Icon class for the dropdown button.
is-disabled boolean False If the buttons are disabled.
disabled-focusable boolean False Set the disabled but focusable state. This must always be used with the 'instructions' attribute.
instructions string False Short description when the button is disabled. It should contain the reason and what needs to be done to enable it. This is for screen readers.
type string False 'emphasized', 'transparent', 'ghost', 'positive', 'negative' and 'attention'. If not specified, normal state is assumed.
callback function False The passed function will be called when the main action button is clicked.
