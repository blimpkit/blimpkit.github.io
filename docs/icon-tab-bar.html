Icon Tab Bar

The Icon Tab Bar can be used for navigation within an object, or as a filter. Each tab of the component links to a different content area or view.

Attributes - Icon Tab Bar

Name Type Required Description
icon-only boolean False When true, tabs can have icons but no text.
has-icons boolean False When true, tabs can have both icons and text.
is-process boolean False When the tabs are used as steps for a wizard, set this to true.
is-nav boolean False When the tabs are used as a main navigation, inside a Shell Navigation.
has-filter boolean False The tab bar has a filter tab.
has-counters boolean False The tabs inside the bar will have counters.
flat-nav boolean False Use with flat navigation.
side-padding string False Size of the side padding. Supported options are 'sm', 'md', 'lg', 'xl', 'xxl' and 'responsive'.
transparent boolean False Transparent background.
translucent boolean False In translucent mode the header gets "--sapObjectHeader_Background" background color and the panel "--sapGroup_ContentBackground" background color.
unfocused boolean False Tabs will be in an unfocused state.
selected-tab-id string True Set selected tab.
compact boolean False Compact mode. Does not work when there are icon tabs.

Attributes - Icon Tab Bar Tab

Name Type Required Description
label string False Tab label.
description string False Description label next to the icon.
counter string False Counter label shown next to or above the label.
has-badge boolean False If the tab has a badge indicator.
icon string False Icon class.
link string False Link.
tab-id string True The id of the tab.
tab-hint string False Show a small text hint next to the label. This is ignored in icon and process mode.
state string False State of the tab. Possible options are 'positive', 'negative', 'critical' and 'informative'.
is-last-step boolean False If the tabs is the last step of a process.
on-close function False Function that will be called when the tab close button is clicked. The tab ID will be passed as a parameter.
closable boolean False Enables the close button next to the tab label, regardless if an onClose function has been provided.
is-hidden boolean False Whether the tab should be visible or not.

Attributes - Icon Tab Bar Filter Item

Name Type Required Description
label string False Filter tab label.
counter string False Counter label shown next to or above the label.
link string False Link.
tab-id string True The id of the tab.
is-selected boolean False If the tab is selected.

Attributes - Icon Tab Bar Overflow

Name Type Required Description
label string True Button label.
state string False State of the button. Options - 'hover', 'active' and 'focus'.
align string False Relative position of the popover. For possible options, look at "fd-popover-body".

Attributes - Icon Tab Bar Overflow Item

Name Type Required Description
label string True Tab label.
counter string False Counter label shown next to the label.
has-badge boolean False If the tab has a badge indicator.
icon string False Icon class.
link string False Link.
tab-id string True The id of the tab.
tab-hint string False Show a small text hint next to the label. This is ignored in icon and process mode.
state string False State of the tab. Possible options are 'positive', 'negative', 'critical' and 'informative'.
on-close function False Function that will be called when the tab close button is clicked. The tab will have an "X" button and on click, the tab ID will be passed as a parameter.

Attributes - Icon Tab Bar Buttons

Name Type Required Description
align-right boolean False Align the buttons to the right of the tab panel.

Attributes - Icon Tab Bar Panel

Name Type Required Description
tab-id string True The id of the tab this panel belongs to.
