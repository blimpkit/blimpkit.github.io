Progress Indicator

The progress indicator component informs the user about an ongoing operation. Compared to the busy indicator, progress is shown in percentages.

Attributes

Name Type Required Description
min-value number False The minimum value of the indicator. Default is 0.
max-value number False The maximum value of the indicator. Default is 100.
current-value number True The current value of the indicator.
label string False The label above the indicator. If not provided, the current value will be used.
state string False State of the indicator. The available options are 'informative', 'positive', 'critical' and 'negative'.
