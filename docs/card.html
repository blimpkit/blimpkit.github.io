Card

The Card is a container that consists of a Header, Extended Header, Numeric Content, Content and Footer area. It takes the width of its parent. An additional Media Block is optional. The Header could be placed either above or below the Content and is not separated with a line. All building blocks are optional.

Attributes - Card

Name Type Description
card-type string This can be 'object' | 'standard' | 'list' | 'table' to indicate the card type
compact boolean Whether to apply compact mode
in-list boolean When the card in inside a list

Attributes - Card Media

Name Type Description
with-padding boolean Adds padding around the image.
link string Link to the image.

Attributes - Card Header

Name Type Description
title string Card header title.
subtitle string Card header subtitle.
interactive boolean Whether the card header is interactive. Defaults to false.
status string Status text.
status-type string One of 'negative', 'critical', 'positive' or 'informative'.
is-counter boolean When the card status is of the 'counter' type.
state string State of the header - 'hover', 'active' and 'focus'.
role-description string ARIA role description for the header.
description string ARIA description for the interactive header.
