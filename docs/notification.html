Notification

Notifications are used to relay information to the user about a situation or task that requires their input.

Attributes - Notification

Name Type Required Description
is-selected string False If the notification should be selected.
is-banner boolean False Show the banner in the top right corner.

Attributes - Notification Title

Name Type Required Description
is-unread string False When true, the tile will be displayed in bold.

Attributes - Notification Icon

Name Type Required Description
bk-notification-icon string True Type of icon. Available options - 'information', 'warning', 'negative' and 'positive'.
