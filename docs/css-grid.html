Maximum number of grid columns

You can set the maximum number of columns in a grid. Use the "bk-grid--cols-{columnNumber}" class (where the {columnNumber} is the number of columns from 1 to 12) in addition to the "bk-grid" class.

Position grid items horizontally

Class Description
bk-grid--justify-items-start Items are placed at the start of each cell.
bk-grid--justify-items-center Items are placed at the center of each cell.
bk-grid--justify-items-end Items are placed at the end of each cell.
bk-grid--justify-items-stretch Items are stretched to fit each cell.

Position grid items against the block axis

Class Description
bk-grid--place-content-start Pack items against the start of the block axis.
bk-grid--place-content-center Pack items in the center of the block axis.
bk-grid--place-content-end Pack items against the end of the block axis.
bk-grid--place-content-between Distribute grid items along the block axis so that that there is an equal amount of space between each row on the block axis.
bk-grid--place-content-around Distribute grid items such that there is an equal amount of space around each row on the block axis.
bk-grid--place-content-evenly Distribute grid items such that they are evenly spaced on the block axis.
