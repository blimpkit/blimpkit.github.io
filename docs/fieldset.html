Field Set

A fieldset is used to give semantic meaning to a group of elements inside a form. Grouping fields together into a fieldset also provides styling and accessibility benefits.

Attributes - Field Set

Name Type Required Description
label string False Title for the legend.

Attributes - Form Group

Name Type Required Description
inline boolean False Form items are displayed horizontally.
label string False Text for the group header.
compact boolean False Header size.
transclude-classes string False CSS classes for the inner 'ng-transclude' container.

Attributes - Form Group Header

Name Type Required Description
compact boolean False Header size.
header-id string True Id for the header element. Maps internally to 'aria-labelledby'.

Attributes - Form Item

Name Type Required Description
horizontal boolean False If true, items will be displayed horizontally.
in-list boolean False Set to true if the form item is in an fd-list-item element.

Attributes - Form Label

Name Type Required Description
colon boolean False Puts a colon at the end of the label.

Attributes - Form Input Message

Name Type Required Description
state string False The type of message. Possible values are 'error', 'information', 'success' and 'warning'. Omit or set to undefined/empty if you want the message to not be shown.
message string False The message text that will be shown.
message-fixed boolean False Message css position will be fixed, allowing for use in dialogs.
