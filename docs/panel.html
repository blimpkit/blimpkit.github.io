Panel

The panel is a container for grouping and displaying information. Panels are responsive and can be collapsed to save additional screen space.

Attributes - Panel

Name Type Required Description
expanded boolean True Whether the panel is expanded or not
fixed boolean False Whether the panel is expandable or not. Defaults to false.
compact boolean False Panel size
expanded-change function False A callback called when the Expand button is clicked

Attributes - Panel Expand

Name Type Required Description
hint string True Description that will be used for the aria label and tooltip hint.
