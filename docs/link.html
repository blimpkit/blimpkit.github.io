Link

A link (also known as hyperlink) is an interactive text element that allows users to navigate to another page or trigger a certain event..

Attributes

Name Type Required Description
is-active boolean False Link is in active mode.
is-focus boolean False Link is in focused mode.
state string False You have five options - 'emphasized', 'subtle' and 'inverted'.
left-glyph string False Icon class for the left link icon.
right-glyph string False Icon class for the right link icon.
