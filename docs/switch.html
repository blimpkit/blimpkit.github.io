Switch

The switch mimics a physical switch, allowing users to set individual features (such as personalization or display settings) to either active or inactive. What the switch controls should be clarified with an inline label, while the state it's currently in should be displayed on the switch itself.

Attributes

Name Type Required Description
switch-id string False ID of the input.
name string False Name of the input (used for form validation).
value boolean False Value of the input. Use this only when you do not use ng-model.
semantic boolean False Show the switch with semantic colors.
with-text boolean False Show "On" and "Off" labels in the switch, instead of icons.
switch-labelledby string True Internally, it maps to "aria-labelledby".

Examples

Switch label $scope.switch = { value: true };
