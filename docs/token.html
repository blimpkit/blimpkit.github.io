Token

Tokens are small items of information (similar to tags) that mainly serve to visualize previously selected items. The tokenizer is the container that handles the tokens. Tokens can be added, removed, selected, or deselected.

Tokenizer is a container that wraps multiple tokens.

Attributes - Token

Name Type Required Description
label string True The text of the token.
compact boolean False Whether the token is compact.
is-selected boolean False Enable selected mode.
is-hover boolean False Enable hover mode.
is-focus boolean False Enable focus mode.
is-readonly boolean False Whether the token is read only.
close-aria-label string False Aria label for the close button.
close-clicked function False A callback called when the close button has been clicked.

Attributes - Token indicator

Name Type Required Description
suffix string False The text after the number of hidden tokens.

Attributes - Tokenizer

Name Type Required Description
compact boolean False Whether the tokenizer is compact
is-focus boolean False Enable the focus indicator.
scrollable boolean False Whether tokenizer is scrollable. When false displays 'more' indicator with the number of the hidden tokens
