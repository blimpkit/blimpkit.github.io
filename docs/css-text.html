Text

You can use those classes to change the size and formatting of text.

Class Description
fd-text Generic text block with no paddings.
bk-preformatted Does not apply any formatting to the text. Preserves the original formatting.
bk-pre-wrap Sequences of white space are preserved. Lines are broken at newline characters, at <br>, and as necessary to fill line boxes.
bk-font--italic Sets the font style to italic.
bk-font--small Sets the font size to small.
bk-font--large Sets the font size to large.
