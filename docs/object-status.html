Object Status

Object Status is a short text that represents the semantic status of an object. It has a semantic color and an optional icon. Typically, the object status is used in the dynamic page header and as a status attribute of a line item in a table.

Attributes

Name Type Required Description
status string False One of 'negative', 'critical', 'positive' or 'informative'
glyph string False Icon class.
text string False Object status text.
clickable boolean False For SPAN elements only
inverted boolean False Inverts the background color
indication boolean False Applies generic indication color. Must be a number between 1 and 8 inclusive
large boolean False Increases the size
truncate boolean False By default, Object Status text goes on multiple lines. For a single line text with ellipsis, set this to true.
