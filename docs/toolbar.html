Toolbar

The toolbar enables the user to change the UI or trigger an action. For example, the user can change views, manipulate data or objects, navigate to another page, perform generic actions, and so on. This component is usually paired with buttons, which are always right-aligned.

Attributes - Toolbar

Name Type Required Description
type string False The type of the toolbar. One of 'transparent', 'auto', 'info' or 'solid' (default value)
compact boolean False Applies compact style to the toolbar and all elements inside it.
has-title boolean False Should be used whenever a title is required. This is incompatible iwht "compact" size.
no-bottom-border boolean False Removes the bottom border of the toolbar
active boolean False Enables active and hover states

Attributes - Toolbar Spacer

Name Type Required Description
fixed-width number|string False The fixed with of the spacer. Could be any valid css width value or number in pixels.
